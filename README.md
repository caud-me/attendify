# Attendify
Version 0.1.7

Online based attendance system

## How to use?
You need
- MySQL community server 8.4
- node.js
- dependencies
  - express
  - mysql2
  - bodyparser
  - ws
    
once installed, type in the terminal in VSCode:

```node server.js```
to test the effectiveness modify the `data.json` and the website should update in real-time.
